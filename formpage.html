PROJECT ONE

TABLEWARE SET

This is a set of tableware that I designed. The 3D modelling was done in Rhinoceros. In it, I created each individual piece arranged them to emulate real life scenarios. After, I brought them into Fusion 360, where I was able to apply textures such as wood, ceramic, steel, and glass.

PROJECT TWO

WEARABLE TECHNOLGY

This is a set of wearable technology, designed to be used while hitting the slopes. Each button displays an aspect of skiing or snowboarding, such as elevation, speed, or distance. Similar to the tableware, I modelled them in Rhino and brought them into Fusion 360 for a render. After, I used Photoshop to apply them in a realistic setting.
